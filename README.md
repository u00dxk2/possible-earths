# Possible Earths

A small editorial gallery of counterfactual-geography scenarios. Each scenario pairs a short essay with before/after equirectangular maps in four views — terrain, satellite, Köppen climate classification, and surface temperature — generated by the third-party tool [World Orogen](https://orogen.studio). The site is a gallery + essays. It is not a tool. Orogen is the tool; we link out to it.

Live: https://possible-earths.onrender.com
Source: https://github.com/u00dxk2/possible-earths

## Stack

- [Astro 5](https://astro.build) + MDX, TypeScript strict mode
- Tailwind CSS with `@tailwindcss/typography`
- Self-hosted variable fonts (Source Serif 4, Inter) via `@fontsource-variable`
- No JavaScript at runtime (the only "interactive" component, `MapComparison`, uses CSS-only radio tabs)
- No analytics, no cookies, no third-party scripts
- Deployed as a static site on Render

## Local development

```bash
nvm use            # picks up .nvmrc → Node 20
npm install
npm run dev        # http://localhost:4321
```

Useful commands:

```bash
npm run build      # production build to dist/
npm run preview    # serve the production build
npm run check      # astro check (TypeScript + content collection schema)
npm run format     # prettier
```

Drafts (`status: 'draft'`) appear in `npm run dev` but are excluded from production builds.

## Scenario production pipeline

Producing a real scenario is a manual, multi-step process. There is no automation — each step is a human decision so the maps stay legible and the counterfactual stays clearly intentional.

1. **Acquire a base equirectangular Earth heightmap.** Use a public DEM such as [NASA SRTM](https://www.earthdata.nasa.gov/data/instruments/srtm) or the [Copernicus GLO-30 DEM](https://spacedata.copernicus.eu/collections/copernicus-digital-elevation-model). Downsample to 2048×1024 or 4096×2048 grayscale PNG. **Do not commit the source DEM to this repo** — record the source URL in the scenario's `sources` frontmatter instead.
2. **Edit in GIMP** (or via Python/PIL for programmatic edits) to express the counterfactual. Save as a 2:1 equirectangular 8-bit grayscale PNG. Soften every edit edge with at least 30–60 px of Gaussian blur — Orogen's climate model responds to gradients, and hard edits produce visibly synthetic Köppen banding.
3. **Open https://orogen.studio/import** in a desktop browser (Chrome or Firefox tested). Upload the modified PNG. Orogen treats it as a 2:1 equirectangular elevation map and runs ocean/land detection.
4. **Run climate** (the auto button). Optionally tweak the temperature offset and precipitation multiplier sliders to bracket the scenario or match the temperature anomaly implied by the relevant paleoclimate literature.
5. **Use _Export All_** to save satellite, Köppen, the heightmap roundtrip, and the land mask in one shot. Re-export terrain-only and temperature individually via the _Inspect_ dropdown. You should end up with eight PNGs: four views × _before_ (the unedited map) and _after_ (the modified map).
6. **Save the eight files** into `public/scenarios/<slug>/` using the exact filenames `{before,after}-{terrain,satellite,koppen,temperature}.png`, plus a `cover.{svg,png,jpg}` and the input PNG as `heightmap-input.png`.
7. **Create or update** `src/content/scenarios/<slug>.mdx` with the frontmatter (validated against the Zod schema in `src/content/config.ts`) and the essay body. Cite at least two primary sources.
8. **Set `status: 'published'`**, commit, push to `main`. Render auto-deploys.

## What this site is and isn't

Orogen's own framing is the right one: the climate model is **scientifically informed, not necessarily physically accurate**. It is a heuristic browser-based simulation — useful for exploring the topology of climate response, not for projecting climate change.

This site is:

- A gallery of pre-rendered Orogen outputs paired with editorial essays.
- A consistent place to read counterfactual-geography scenarios alongside the same four map views.
- A way to download the modified heightmaps so anyone can rerun a scenario in Orogen themselves.

This site is not:

- A research tool or a peer-reviewed publication.
- A source of climate projections. For real projections of future Earth climate, see the [Copernicus Climate Data Store](https://cds.climate.copernicus.eu/) and [CMIP6](https://wcrp-cmip.org/cmip-overview/).
- A substitute for the paleoclimate literature it cites for past-Earth scenarios. Where we visualize a past configuration, the maps illustrate that literature — they do not replace it.
